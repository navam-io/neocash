NeoCash is a personal wealth manager which enables users to research and learn about taxation, investing, and personal wealth management as well as manage their tax filing, investment portfolio, and accounts with financial institutions all in once place. NeoCash is designed with privacy-first, data security principles. NeoCash is a web app which runs locally, all data and files stays local and easily available like data and files on user's private folders on their laptop. NeoCash uses latest capabilities offered by Anthropic models and intelligently leverages the best model for a specific task to optimize quality, latency, and performance.

NeoCash frontend is a clone of Claude AI chat interface to provide familiarity for Claude AI users. The interface is suitably modified for NeoCash purpose like suggested prompt reflect the NeoCash personal wealth management topics and prompts. Otherwise it is a near perfect clone of Claude AI chat interface and features as these apply to NeoCash use case.

Scrape https://claude.ai/new/ and visit https://claude.ai/new to clone the Claude chat frontend. The frontend should work exactly like Claude AI and use the Anthropic API Key in .env.local to make the LLM calls.
